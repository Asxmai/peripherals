通信接口

51 项
📡

USART

通用同步/异步收发器 · 1970s

在UART基础上增加同步时钟模式，可作SPI主设备。支持ISO 7816智能卡、IrDA、LIN等协议模式。

→
🔗

I2C

集成电路间总线 · 1982

半双工同步串行总线，SDA/SCL两线，多主多从寻址(7/10位)。标准100kHz，快速400kHz，高速3.4MHz。

→
⚡

SPI

串行外设接口 · 1979

高速全双工同步串行总线，主从架构，SCK/MOSI/MISO/CS四线。CPOL/CPHA四种模式，速率可达数十MHz。

→
🚀

I3C

改进型集成电路间总线 · 2018

MIPI定义的I2C下一代标准。速率12.5MHz+，支持带内中断(IBI)、动态地址分配、HDR模式。向下兼容I2C设备。

→
🚗

CAN 2.0

控制器局域网 · 1991

差分信号多主通信，消息仲裁机制，抗干扰强。CAN 2.0A(11位ID)/2.0B(29位ID)，速率1Mbps。

→
🌐

Ethernet MAC

以太网MAC · 1983

IEEE 802.3以太网控制器，10/100Mbps(RMII/MII)。需外接PHY。支持TCP/IP协议栈。

→
🏭

RS-485

差分通信 · 1983

差分信号长距离通信物理层，支持多点(32/128/256节点)，距离可达1200m。常配合UART使用。

→
💾

QSPI

四线串行外设接口 · 2010s

扩展SPI，4位数据线并行传输，支持XIP(就地执行)。速率可达数百MHz。

→
📮

UART

通用异步收发器 · 1960s

全双工异步串行通信，TX/RX两根线，可配置波特率。最基础的点对点串行通信外设。

🚌

CAN FD

灵活数据速率CAN · 2012

CAN的增强版本，数据段速率可达8Mbps，有效载荷最大64字节。兼容CAN 2.0网络。

🔌

USB

USB设备/主机/OTG · 1996

通用串行总线，Full Speed(12Mbps)/High Speed(480Mbps)。支持Device/Host/OTG模式。

→
🔗

OSPI

八线SPI · 2017

8位数据线并行传输，速率可达数百MHz甚至GHz级。支持DDR模式，带宽是QSPI的两倍。

→
💿

SDIO/SDMMC

SD卡/SDIO接口 · 2000

安全数字接口，支持SD/MMC/eMMC存储卡和SDIO外设(WiFi等)。4位总线，50MHz。

→
🎵

I2S

集成电路间声音 · 1990s

专用音频串行总线，3线(SCK/SD/WS)或4线(+MCLK)。标准飞利浦格式，支持主/从模式。

→
🎶

SAI

串行音频接口 · 2012

高灵活度音频串行接口，支持I2S/左对齐/右对齐/TDM多通道等多种协议。独立时钟和帧同步。

→
🚙

LIN

本地互联网络 · 1999

低成本单线串行总线，主从架构，速率最高20kbps。用于低速子网络。

→
📱

IrDA

红外通信 · 1993

红外串行通信，IrDA SIR标准，通过红外LED实现短距离无线数据传输。

→
🎧

SPDIF

S/PDIF接收/发送 · 1987

索尼/飞利浦数字音频接口标准，支持光纤或同轴传输。采样率最高192kHz。

🔧

MDIO/SMI

管理数据输入输出 · 1983

IEEE 802.3定义的PHY管理总线(MDC/MDIO两线)，用于读写以太网PHY寄存器。

💳

ISO 7816

智能卡接口 · 1987

符合ISO 7816-3标准的智能卡通信接口，支持半双工、字符协议T=0和T=1。

🔋

SMBus

系统管理总线 · 1995

基于I2C的简化协议，用于系统管理通信。定义标准命令和消息格式。

🌡️

1-Wire

单线总线 · 1990s

Dallas/Maxim单线协议，仅一根数据线+地线。每个设备有唯一64位ID。速率约16kbps。

📊

DFSDM

数字滤波器Σ-Δ调制器 · 2017

片内数字滤波器，直接连接外部Σ-Δ调制器。支持FIR/IIR滤波、过采样、多通道级联。

🎤

PDM

脉冲密度调制输入 · 2010s

PDM数字麦克风专用接口，1位高速数据流，片内转换为PCM数据。支持立体声双通道。

🖥️

LVDS

差分信号显示接口 · 1994

低电压差分信号，高速低EMI。用于大尺寸TFT LCD屏驱动、高速数据采集板间通信。

📐

更多...

SSI / BiSS / HART / IO-Link / PMBus / AS-i / SENT / DALI / DMX512 / FlexRay / MOST / PSI5 / DSI3 / CAN XL / SPMI / CEC / EnDat / Tamagawa / LPUART / LPSPI / LPI2C / FlexIO / SGPIO / M-LVDS / Wiegand / PSSI

通信接口分类下还有25种以上外设协议，涵盖工业、汽车、音频、安防等领域。

⏱️

定时器/计数器

12 项
⏱️

TIM 通用定时器

通用定时器 · 1990s

多功能16/32位定时器，支持输入捕获、输出比较、PWM、编码器接口、单脉冲。
🕐

SysTick

系统定时器 · 2004

Cortex-M内核24位递减计数器，与主频同步。常用于RTOS时基。

📆

RTC

实时时钟 · 1990s

独立域低功耗时钟，LSE(32.768kHz)驱动。日历、闹钟、周期唤醒。VBAT备份。

🐕

IWDG

独立看门狗 · 1990s

独立硬件看门狗，LSI时钟驱动。超时未喂狗触发系统复位。

⚙️

Advanced TIM

高级定时器 · 2007

在通用定时器基础上增加互补输出、死区控制、刹车功能。专用于电机控制。

🔁

更多...

Basic TIM / LPTIM / WWDG / HRTIM / FTM / CMT / STU/TMU

包括基本定时器、低功耗定时器、窗口看门狗、高分辨率定时器(HRTIM)等7种定时器外设。

📊

模拟外设

14 项
📊

ADC

模数转换器 · 1990s

模拟电压转数字值。单端/差分、多通道扫描、注入通道、硬件过采样。12/16位，数MSPS。
📈

DAC

数模转换器 · 1990s

数字值转模拟电压。多通道、波形生成、DMA。12位，微秒级建立时间。
⚖️

COMP

模拟比较器 · 1990s

比较两路模拟电压大小，输出数字结果。可编程迟滞、低功耗。百纳秒响应。

🔊

OPAMP

运算放大器 · 2010s

片内集成运放，可配置增益。输入输出映射到GPIO。减少外部器件。

🌡️

更多...

TS_CAL / Bandgap / Analog MUX / S&H / Current Source / PGA / VREFBUF / Σ-Δ / Charger / EMC Filter

还包括温度传感器、带隙基准、模拟多路复用器、采样保持电路、可编程电流源等10种模拟外设。

🔔

中断与DMA

5 项
🔔

NVIC

嵌套向量中断控制器 · 2004

Cortex-M内核中断管理器，可配置优先级(抢占+子优先级)、尾链机制。

⚡

EXTI

外部中断/事件控制器 · 2007

监测GPIO边沿(上升/下降/双边沿)，触发中断或事件。支持滤波、软件触发。
🔄

DMA

直接存储器访问 · 1990s

外设与内存间高速传输，无需CPU参与。多通道/多流、优先级仲裁、循环模式。

🔀

DMAMUX

DMA复用器 · 2017

DMA请求源灵活映射到任意DMA通道，突破固定映射限制。支持同步/外部触发。

🧠

SmartDMA/MDMA

智能DMA · 2019

增强型DMA，支持内存到内存(MDMA)、链表描述符、硬件请求复用、数据预取。

💾

存储接口

7 项
💾

FLASH

Flash控制器 · 1990s

管理内部Flash读写、擦除、加密、安全区域(ROP)。支持双Bank并行读取。

🗄️

FMC/FSMC

灵活存储控制器 · 2007

外部SRAM/PSRAM/NOR/NAND/SDRAM并行总线接口。16位数据，地址/数据复用或非复用。

📦

XSPI

外部存储器接口 · 2018

支持NOR/RAM/PSRAM的串行/并行接口。DDR模式、加密传输。

📝

更多...

EEPROM Emul / ART / TCM / SDRAM Ctrl

还包括EEPROM模拟、Art加速器、紧密耦合存储器(TCM)、SDRAM控制器等4种存储外设。

🕐

时钟与复位

9 项
🕐

RCC

时钟控制单元 · 1990s

管理所有时钟源(HSI/HSE/LSI/LSE/PLL)和总线时钟分配。外设时钟使能/复位均通过RCC。

⚡

HSI / HSE

内部/外部高速振荡器

HSI: 片内RC高速振荡器(16/48/64MHz)。HSE: 外接晶振(4~26MHz)，作为系统主时钟或PLL源。

🔄

PLL

锁相环 · 1990s

将低频时钟倍频到更高频率。主PLL用于系统时钟，专用PLL用于USB(48MHz)/SAI。

🔧

更多...

LSI / LSE / CSS / MCO / MSI

还包括内部/外部低速振荡器、时钟安全系统、时钟输出、内部多速振荡器等6种时钟外设。

🔋

电源管理

6 项
🔋

PWR/PMU

电源管理单元 · 1990s

管理低功耗模式(Sleep/Stop/Standby/Shutdown)。控制电压调节器、备份域、唤醒源。

⚡

PVD

可编程电压监测 · 1990s

监测VDD与可编程阈值比较。低于阈值触发中断或复位。多级阈值。

🔌

BOR

欠压复位 · 1990s

当VDD低于固定阈值时自动硬件复位。防止低电压下程序异常运行。

🔧

更多...

OTP/FUSE / BKP/BBR / VBAT Domain

还包括熔丝位/选项字节、备份寄存器、电池备份域等3种电源管理外设。

🔒

安全与加密

12 项
🔒

CRC

CRC计算单元 · 2000s

硬件CRC校验，支持多种多项式(CRC32等)。快速数据校验。

🎲

RNG

真随机数发生器 · 2000s

基于模拟噪声源的真随机数。通过AHB接口读取。

🔐

CRYP/AES

加密加速器 · 2012

硬件加密引擎，DES/TDES/AES(ECB/CBC/CTR/GCM)。高速加解密。

🛡️

更多...

HASH / Secure Boot / MPU / OTP / TrustZone / TAMP / PUF / HSM / Fault Injection

还包括哈希处理器、安全启动、内存保护单元、TrustZone、物理不可克隆函数(PUF)等9种安全外设。

🖥️

显示与摄像头

10 项
🖥️

LTDC

LCD-TFT控制器 · 2012

TFT显示控制器，RGB接口。多层叠加、Alpha混合、色深转换、抖动。

📷

DCMI/DVP

数字摄像头接口 · 2010

并行数字摄像头接口，8/10/12/14位数据总线。支持同步码、硬件JPEG解码。

🎨

DMA2D

图形加速器 · 2012

2D图形加速引擎，颜色转换、Alpha混合、缩放/旋转。DMA操作帧缓存。

📱

更多...

MIPI DSI / MIPI CSI-2 / LCD Driver / DPI / Intel 8080 / ePaper IF / DCIC

还包括MIPI DSI主机、MIPI CSI-2摄像头、段式LCD驱动、并行RGB/DPI、电子纸接口等7种显示外设。

🔧

调试与测试

9 项
🔧

SWD

串行调试接口 · 2004

仅需SWDIO/SWCLK两根线。支持断点、单步、内存读写、Flash下载。

🔌

JTAG

JTAG调试接口 · 1990

IEEE 1149.1标准，5线。支持边界扫描和调试。

📡

SWO/ITM

串行跟踪输出 · 2004

通过SWO引脚输出调试信息。printf重定向、变量跟踪、时间戳。

🔬

更多...

ETM / ETB / TPIU / Boundary Scan / LBIST / MBIST

还包括嵌入式跟踪宏单元、跟踪缓冲器、跟踪端口接口、逻辑/存储内建自测等6种调试外设。

人机交互

5 项
👆

GPIO

通用输入输出 · 1990s

可配置数字I/O引脚，支持推挽/开漏/上拉/下拉/模拟/复用功能。GPIO中断。
🖐️

TSC

触摸感应控制器 · 2012

电容式触摸感应，多通道触摸按键/滑条/转盘。可配置采样/阈值/防抖。低功耗。

💡

LED Ctrl

LED控制器 · 2015

专用LED驱动控制器，支持PWM调光、呼吸效果、闪烁模式。硬件自动运行不占CPU。

⌨️

更多...

KBD Matrix / LCD Boost

还包括键盘矩阵接口、LCD升压转换器等2种人机交互外设。
